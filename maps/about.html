Qu'est-ce que ShardUI 2 Maps ?

ShardUI 2 Maps est une plateforme de cartographie dédiée aux serveurs Minecraft. Elle permet aux administrateurs et joueurs de visualiser leurs mondes de manière interactive et intuitive.

Basée sur les technologies web modernes (Leaflet, Tailwind CSS, DaisyUI), elle offre une expérience utilisateur fluide et responsive sur tous les appareils.

Technologies Utilisées

Leaflet Cartographie interactive
Tailwind CSS Styling moderne
DaisyUI Composants UI
FontAwesome Icônes
